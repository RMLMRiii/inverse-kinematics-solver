# Inverse Kinematics Solver

A Python tool for solving 2D inverse kinematics for a two-link robotic arm.

## Features
- Solves for joint angles from a target x-y position
- Uses two-link robotic arm geometry
- Organized function-based structure
- Includes sample target position data
- Separate target position viewer

## Technologies Used
- Python
- CSV file handling
- Trigonometry
- Basic robotics kinematics

## How to Run
Run the script in Python:

python inverse_kinematics.py

To view the sample target data:

python target_viewer.py
